# Dark Souls Progression

"What could I do next? No spoilers!"

This app only shows things directly connected to where you tell us you've already been.

The website for this app is live at: https://dark-souls-playground.github.io/progression

## Running this locally

- Requires Node 14, Angular 12/14
- `ng serve`

Originally generated with [Angular CLI](https://github.com/angular/angular-cli) version 12.2.12.
